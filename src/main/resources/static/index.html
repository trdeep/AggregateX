AggregateX

基于领域驱动设计（DDD）的现代化 Java 框架

JDK 21+ Spring Boot 3.5 Version 1.0.0 MPL-2.0 License

🚀 核心特性

AggregateX 整合了 CQRS 模式、事件溯源、事件总线等 DDD 核心概念，为开发复杂业务系统提供完整的架构支持。

🏗️ 完整的DDD分层架构

提供标准的领域驱动设计分层架构，确保代码结构清晰、职责明确。

  • 领域层：聚合根、实体、值对象、领域事件
  • 应用层：命令处理、查询处理、应用服务
  • 基础设施层：持久化、消息、外部服务集成

⚡ CQRS模式实现

读写分离的命令查询职责分离模式，提升系统性能和扩展性。

  • 命令处理：状态变更、数据验证、事务管理
  • 查询处理：优化读取、灵活查询、数据组装
  • 并发控制：确保数据一致性

🛠️ 智能代码生成器

内置强大的DDD模块代码生成器，快速搭建标准化的领域模块。

  • 一键生成完整DDD模块结构
  • 基于FreeMarker的可定制模板
  • 交互式命令行界面
  • 标准化代码规范

📡 事件驱动架构

完善的事件总线和事件溯源机制，支持微服务间的异步通信。

  • 领域事件发布与订阅
  • 事件溯源存储
  • 事件重放机制
  • 分布式事件处理

🔧 技术栈

Java 21 Spring Boot 3.5 Spring Data JPA FreeMarker Maven Gradle JUnit 5 SLF4J

🎯 适用场景

💼 复杂业务系统

适用于业务逻辑复杂、需要严格建模的企业级应用系统。

🔄 微服务架构

为微服务提供标准化的DDD实现，确保服务间的一致性。

📈 高并发场景

通过CQRS和事件驱动，支持高并发、高性能的业务处理。
